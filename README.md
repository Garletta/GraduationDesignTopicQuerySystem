# GraduationDesignTopicQuerySystem
Graduation Design Topic Query System

## 使用SSH框架

### 毕业设计题目查询系统  
* 用户登录、  
* 毕业设计题目所属学院录入，  
* 毕业设计题目录入，  
* 毕业设计题目信息查询（支持模糊查询）。

# GraduationDesignTopicQuerySystem
毕业设计题目查询网页  

## 使用SSH框架
* Hibernate5  
* Struts2.5  
* Spring4.2  

### 毕业设计题目查询系统  
* 用户登录、  
* 毕业设计题目所属学院录入，  
* 毕业设计题目录入，  
* 毕业设计题目信息查询（支持模糊查询）。
